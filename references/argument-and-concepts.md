# 论证骨架与概念纪律

## 一、先确定论文正在证明什么

把研究对象、核心问题和中心判断分开书写。

- 研究对象说明考察什么材料和关系。
- 核心问题说明现有解释为何不足。
- 中心判断说明全文最终能够证明到什么程度。
- 研究价值说明这一判断增加了什么解释能力，不等于罗列“理论意义、现实意义”。

中心判断应当能够被拆解为若干需要分别证明的子判断。每章至少承担一个子判断，避免一章同时完成概念史、方法论、历史发生、机制分析和当代应用。

## 二、建立论文主线

用下列四项检查全文骨架：

1. **起点**：读者为什么需要研究这个问题。
2. **边界**：研究对象同相邻概念怎样区分。
3. **机制**：对象如何形成、运行和再生产。
4. **效力**：结论能够解释什么，不能推出什么。

绪论开头通常按“现实关系—现有理论能解释什么—尚未解释什么—本文问题”推进。不要从抽象概念史突然跳到研究问题，也不要把“术语出现频率变化”直接当作理论问题已经成立的理由。

## 三、给每章定义有限功能

为每章填写以下矩阵：

| 项目 | 内容 |
|---|---|
| 输入 | 上一章已经确立的前提 |
| 问题 | 本章必须回答的一个核心问题 |
| 证据 | 需要细读的原始文本和辅助研究 |
| 输出 | 本章能够确立的有限结论 |
| 边界 | 明确留给下一章的问题 |

章节标题应当反映实际功能。若标题声称讨论“生成逻辑”，正文却只说明成立条件，应调整标题或补足论证。

相邻章节重复时，不按字面删重，先区分功能。例如：

- 思想史背景说明问题从哪里来，概念限定说明本文怎样使用它。
- 历史前提说明关系如何形成，范畴分析说明这种关系如何被理论把握。
- 生产机制说明支配如何实现，当代章节说明同一结构在新条件下如何变化。
- 结论集中陈述已证明的结构，不按章节顺序重新复述全文。

## 四、保持历史过程与叙述方法的区别

理论叙述可以先分析简单规定，再重建复杂总体；现实历史并不会因此由简单概念自行生成。

检查以下风险：

- 把若干相邻范畴排列成自动推演链条，仿佛一个概念必然产生下一个概念。
- 把范畴在书中的次序当作历史事件的发生次序。
- 把资本与劳动的结合写成无冲突的逻辑完成。
- 把一般规定提升为规范尺度，再用现实是否符合它判定异化。

更稳妥的写法是说明：历史关系先形成，范畴在特定社会总体中获得现实有效性，理论通过抽象和叙述把这些关系揭示出来。

## 五、建立概念登记表

每个核心概念至少记录：

| 字段 | 要求 |
|---|---|
| 名称 | 全文统一的中文术语 |
| 原词 | 首次实质界定时使用的外语词 |
| 工作定义 | 当前论文中的有限含义 |
| 相邻概念 | 容易混淆的术语 |
| 文本根据 | 原始文本或权威研究 |
| 允许表达 | 后文可使用的简称 |
| 禁止替换 | 会造成对象滑移的说法 |
| 首次位置 | 正文首次解释所在章、节 |

不要把作者自己的综合解释写成学界已经公认的固定分类。使用“本文据此理解”“这里所指”时，应确有解释责任；不要每段都以“本文认为”开头。

概念第一次出现可以只做简要定位，第一次进入实质论证时必须解释。解释完成后保持名称连续，不在同一理论路径的不同命名、概念名词与日常动词、上位概念与具体分析对象之间随意切换。

## 六、区分一般规定与批判判断

一般规定通常说明活动得以发生的条件，不能直接承担批判尺度。对象化、劳动过程的一般要素或人的社会能力，不因其一般性便自动证明现实关系是异化的。

批判判断应当说明：

- 行动主体同活动条件处于何种现实关系；
- 主体形成的成果或共同力量如何取得独立形式；
- 这种形式怎样返回并规定主体；
- 关系如何通过制度、所有权、时间尺度或组织方式得到维持。

批判根据可以从对象自身的社会形式、主体位置和运行后果中取得，但不要随意发明“三种内在性”“三组矛盾”等分类。若采用分类，必须说明来源或证明其分析必要性。

## 七、按段落检查推进

一个有效段落通常完成一种动作：

- 提出需要解释的困难；
- 引入并解释文本证据；
- 区分两个相邻概念；
- 限定一个过强判断；
- 说明历史条件或运行机制；
- 完成章节接口。

段落可以包含多句，但核心判断不宜不断变化。引用、转述和作者分析要在句法上区分清楚。段末过渡应从已经得到的结论自然引出未解决的问题，不写“下一节将讨论”“本章到这里完成了”。

## 八、限制结论强度

优先使用能够准确表达证据范围的限定：

- “主要表现为”“在这一语境中”“提供了条件”“构成一项前提”；
- “不能单独说明”“尚不足以推出”“在特定条件下可能转化为”；
- “相关研究显示”“在所考察的案例中”。

谨慎使用“根本”“必然”“彻底”“绝对”“完全”“决定性”“无可争辩”。强断言只有在文本和论证都足够明确时保留。
