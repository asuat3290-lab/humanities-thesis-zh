# 马克思文本与 MEGA² 工作流

仅在论文涉及马克思原始文本、MEGA²、德文术语或版本考证时加载。

本文件是可选领域 profile，不属于默认规则；无关任务不得据此改变概念、引文或语言检查。

## 一、区分文本地位

使用文本前确认：

- 公开发表的著作还是未刊手稿；
- 作者生前定稿、编辑整理稿、摘录笔记或边栏批注；
- MEGA² 所属部门、卷册和页码；
- 中文译本的版本和译文；
- 该段文字在原有论证中的位置。

手稿可以揭示概念实验、论证过程和未收入定稿的分析，不能自动等同于最终发表文本。某段未收入正文，也不能仅凭这一事实断定其中问题已被放弃。

## 二、检索顺序

1. 阅读本地工作台或检索程序的说明文件。
2. 理解语料库目录、OCR 命名、卷册映射和页码字段。
3. 用德文关键词、词形变化和相邻概念检索。
4. 打开命中位置的前后页，恢复段落语境。
5. 对照正式 MEGA² 页码和必要的中文译本。
6. 记录检索词、卷册、页码、上下文和拟支持判断。

若存在 mega_ocr_pipeline.py 或 RAG 检索程序，先读程序和说明，不猜测参数。OCR 只用于定位。德文引文和页码必须回到可核验页面确认。

## 三、引用分析顺序

按以下顺序写：

1. 原文关键句；
2. 概念和句法解释；
3. 该段在马克思论证中的位置；
4. 它怎样支持当前章节；
5. 它如何导向后续问题；
6. 它不能证明什么。

不要以引文长度代替分析。大段引用应当具有不可被简短转述替代的文本价值。

## 四、关键术语边界

| 中文术语 | 常见原词 | 使用提醒 |
|---|---|---|
| 异化 | Entfremdung | 不等同于一切疏离感或社会失范 |
| 外化、外在化 | Entäußerung | 结合具体语境，不与 Entfremdung 自动互换 |
| 对象化 | Vergegenständlichung | 活动形成对象性成果的一般过程，不自动等于异化 |
| 物化 | Verdinglichung | 社会关系取得物的形式；不与对象化或拜物教混同 |
| 拜物教 | Fetischismus | 社会关系表现为物的自然属性及其客观形式 |
| 劳动一般 | Arbeit überhaupt | 现代范畴的历史有效性，不充当超历史规范尺度 |
| 抽象劳动 | abstrakte Arbeit | 在价值形式中的社会规定，不化约为纯生理耗费 |
| 劳动过程 | Arbeitsprozeß | 撇开特定社会形式考察的一般规定 |
| 劳动力 | Arbeitskraft | 同已经实现的劳动区分 |
| 劳动能力 | Arbeitsvermögen | 注意不同手稿和译本的语境 |
| 价值增殖过程 | Verwertungsprozeß | 同价值形成过程及劳动过程区分 |
| 活劳动 | lebendige Arbeit | 正在实现的劳动活动 |
| 对象化劳动 | vergegenständlichte Arbeit | 不包揽全部知识、文化和社会能力 |
| 从属 | Subsumtion | 不简化为日常意义的“服从” |
| 形式从属 | formelle Subsumtion | 资本在既有劳动方式上确立支配 |
| 实际从属 | reelle Subsumtion | 资本重组劳动过程；具体中介须分析 |
| 一般智力 | general intellect | 不直接等同于人工智能或固定资本 |

首次实质界定可用“中文（德文，英文）”。后文保持中文术语，不反复括注。引用中出现的词形应服从原文，不强行改成登记表形式。

## 五、避免重新本质化劳动

劳动一般、劳动过程的一般规定和对象化不能充当理想劳动模型。异化劳动不能按“劳动一般具有某规定—现实劳动偏离该规定—现实劳动是异化劳动”的方式推出。

更稳妥的论证从现实关系展开：

- 生产者同生产条件分离；
- 劳动力成为商品；
- 劳动过程进入价值增殖过程；
- 资本通过制度、时间和技术组织劳动；
- 产品、生产资料和共同生产力取得资本形式；
- 对象化力量作为继续支配活劳动的条件返回生产。

批判力量来自劳动者自身力量在资本主义关系中的独立化和反向支配，不需要预设内容完备的本真劳动。

## 六、谨慎处理思想发展

避免用“青年马克思/成熟马克思”的绝对对立预设结论。可以比较问题语言、范畴结构、历史规定和论证方式，但应承认连续性与断裂性仍有争议。需要限定时期时，优先使用“《资本论》时期”。

术语减少、问题变化和理论放弃是三个不同判断。词频只能构成线索，不能单独证明思想史结论。

## 七、脚注格式

确认卷册和页码后，可使用：

- 参见 MEGA² II/5, S. 122。
- 参见 MEGA² II/4.1, S. xxx。

无法确认页码时不要把 xxx 交付。把不确定项留在待核验清单，正文脚注保持最简，不写检索过程。

中文页码只有在最终采用的版本中核对后再写。不要从 MEGA² 页码推测中文版页码。

## 八、知识与对象化劳动

工具、机器和建筑可以较严格地视为既往劳动形成的生产资料。科学知识、工艺经验和社会协作还可能形成于研究、教育、交往和文化积累。应说明这些能力如何被吸收、固定在机器、标准、程序或组织制度中，而不是把所有知识直接称为对象化劳动。

人工智能分析还需区分：

- 企业组织或购买的劳动；
- 研究、工程和维护劳动；
- 用户活动形成的数据；
- 长期积累的公共知识和文化材料。

用户数据可以成为资本占有的资源，不等于用户自动成为雇佣劳动者或价值生产劳动者。
